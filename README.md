Soul Mutt Matcher 2026

An interactive web application that helps potential dog owners find a breed that matches their lifestyle using real-world AKC data.

## ✨ Features
- **Lifestyle Matching:** Uses sliders for energy, shedding, and trainability.
- **Smart Filters:** Filter by weight/size and "Kid-Friendly" demeanor.
- **Breed Mixer:** Experiment with trait-averaging for mixed breeds.
- **Adopt Don't Shop:** Generates direct links to local shelter searches for your match.

## 🛠️ Tech Stack
- **Language:** Python 3.x
- **Framework:** [Streamlit](https://streamlit.io)
- **Data Source:** AKC Breed Database (via Kaggle)
- **Library:** Pandas for data manipulation

## 🚀 How to Run Locally
1. Clone this repo.
2. Install requirements: `pip install -r requirements.txt`
3. Run the app: `streamlit run app.py`
